 # VeoVision Frontend

Web UI to view processed clips (regular and famous). Uses the `_browser.mp4` files generated by the converter for smooth playback.

## Quick Start

1) Start the server (range-support, port 5600):
```bash
python start_video_server.py
```
2) Open in browser:
```
http://localhost:5600/veo_frontend/
```

## If videos don't play
Convert processed videos to browser-safe H.264/AAC copies (keeps originals, adds `_browser.mp4`):
```bash
python convert_videos_for_browser.py
```

## Expected files
- Regular clips: `regular_clips/data_content/*_browser.mp4`
- Famous clips:  `famous_clips/data_content/*_browser.mp4`

## Layout
- Regular: 3-up modal (main, 2D tactical, heatmap)
- Famous: 1-up modal (player detection)

## Theme
Black primary, green/yellow accents, VeoVision logo at top.
